# Вторая жизнь — сайт приюта для животных

Сайт-визитка приюта «Вторая жизнь». Проект выполнен в рамках учебной практики.

## Страницы сайта
- Главная
- О нас
- Помощь
- Новости
- Контакты
- Животные (с фильтрацией)

## Функционал
- Фильтрация карточек животных (собаки, кошки, котята и щенки)
- Карусель новостей на главной странице
- Форма обратной связи с сохранением заявок в базу данных
- Раскрывающийся блок FAQ
- Карта Яндекс.Карт на странице контактов

## Технологии
- HTML, CSS, JavaScript
- PHP (обработка формы)
- MySQL (база данных)

## База данных
- Имя БД: shelter_db
- Таблица: applications
- Поля: id, name, phone, email, message, created_at

## Как запустить
1. Установите OpenServer или XAMPP
2. Поместите папку проекта в domains (OpenServer) или htdocs (XAMPP)
3. Запустите сервер (Apache и MySQL)
4. Импортируйте базу данных shelter_db через phpMyAdmin
5. Откройте в браузере: http://localhost/практика/index.html
